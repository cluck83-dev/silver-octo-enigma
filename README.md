# silver-octo-enigma

A progressive web app (PWA) for learning Hebrew vocabulary with German translations.

## Disclaimer

This project is a personal learning tool created for private and educational use.  
It is not affiliated with, endorsed by, or connected to any publishers, authors, or official learning materials.

All vocabulary data is used solely for learning purposes.  
If you are a copyright holder and have any concerns, please contact me.

---

## Features

- 📚 Learn vocabulary in structured units
- 🎯 Multiple learning modes (by lesson, by word type, mixed)
- 🔄 Interactive flashcards (Hebrew ↔ German)
- 🔊 Audio pronunciation support
- 📱 Installable as a mobile app (PWA)
- 🚫 Offline usage possible
- 🐛 Simple error reporting

---

## Installation (Mobile)

### iOS (iPhone / iPad)

1. Open the app URL in Safari  
2. Tap the share icon  
3. Select **"Add to Home Screen"**  
4. Confirm  

---

### Android

1. Open the app URL in Chrome  
2. Tap the menu (⋮)  
3. Select **"Install App"** or **"Add to Home Screen"**  
4. Confirm
